# ระบบคะแนนค่ายเยาวชนรักพงไพร

เว็บนี้เป็นต้นแบบพร้อมใช้สำหรับ GitHub Pages มีแดชบอร์ดคะแนนรวม หน้าแอดมิน หน้าให้ดาวจาก QR และแกลอรี่ภาพ

## เริ่มทดลอง

เปิดไฟล์ index.html ได้ทันที ข้อมูลทดลองจะเก็บในเครื่องผ่าน localStorage

- รหัสแอดมิน: 9999
- รหัสครูฐานสำรวจพืช: 1234
- รหัสครูฐานรอยเท้าสัตว์: 2468

## วิธีใช้งานหลัก

1. เข้าเมนูตั้งค่า
2. กรอกรหัสแอดมิน 9999
3. สร้างกลุ่มและฐานกิจกรรม
4. พิมพ์ QR ประจำกลุ่ม
5. ครูสแกน QR แล้วกรอกรหัสครู 4 หลักเพื่อให้ดาว
6. หน้าคะแนนรวมจะแสดงอันดับและคะแนนของแต่ละกลุ่ม

## ขึ้น GitHub Pages

1. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้เข้า GitHub repository
2. เปิด Settings > Pages
3. เลือก branch ที่ใช้เผยแพร่
4. ใช้ URL GitHub Pages เป็นลิงก์หลักของค่าย

## ต่อ Google Sheet

1. สร้าง Google Sheet ใหม่
2. เปิด Extensions > Apps Script
3. วางโค้ดจาก google-apps-script.gs
4. Run ฟังก์ชัน setup หนึ่งครั้ง
5. Deploy > New deployment > Web app
6. ตั้งค่า Execute as: Me และ Who has access: Anyone with the link
7. คัดลอก Web app URL มาใส่ใน config.js ตรง apiUrl

## หมายเหตุเรื่องรูปภาพ

ต้นแบบนี้แสดงรูปจากเครื่องผู้ใช้ในโหมดทดลองก่อน หากต้องการเก็บรูปจริงบน Google Drive ให้เพิ่มส่วนรับ base64 ใน Apps Script แล้วบันทึก URL กลับลงชีต Photos
